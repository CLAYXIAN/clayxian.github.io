动态规划

已知问题规模为n的前提A，求解一个未知解B

此时，如果把问题规模降到0，即已知A0，可以得到A0->B

采用动态规划求解的问题一般要具有3个性质：

  1. 最优化理论：如果问题的最优解所包含的子问题的解也是最优的，就称该问题具有最优子结构，即满足最优化原理。
  2. 无后效性：即某一阶段状态一旦确定，就不受这个状态以后决策的影响。也就是说，某状态以后的过程不会影响之前的状态，只与当前状态有关。
  3. 有重叠子问题：即子问题之间是不独立的，一个子问题在下一阶段决策中可能被多次用到。()

动态规划的一般思路：

动态规划所处理的问题是，一般由初始状态开始，一般由初始状态开始，通过对中间阶段决策的选择，到达结束状态。这些决策形成了一个决策序列，同时确定了完成整个过程的一条活动路线（通常是最优的活动路线）。动态规划的设计都有着一定的模式，一般要经历以下几个步骤。

​

  1. 划分阶段：按照问题的时间或者空间特征，把问题风车若干个问题。在划分阶段时，注意，否则问题就无法求解。
  2. 确定状态和状态变量：将问题发展到格格阶段时所处的各种客观情况用不同的状态表示出来。当然，状态的选择要满足无后效性。
  3. 确定决策并写出状态转移方程：因为决策和状态转移有着天然的联系，。所以如果确定了决策，状态转移方程也就可写出。但事实上常常是反过来做，
  4. 寻找边界条件：给出的状态转移方程是一个递推式，需要一个递推的终止条件或边界条件。

一般，只要解决问题的和确定了，就可以写出

实际应用中可以按以下几个简化的步骤进行设计：

  1. 分许最优解的性质，并刻画其结构特征。
  2. 递归的定义最优解。
  3. 以自底向上或自顶向下的记忆化方式（或备忘录法）计算出最优值。
  4. 根据计算最优值时得到的信息，构造问题的最优解。

算法实现的说明

动态规划的主要难点在于理论上的设计，也就是上面四个步骤的确定，一旦设计完成，实现部分就会非常简单。

使用动态规划求解问题，就是确定：

递推关系必须是从次小的问题开始到较大的问题之间的转化，从这个角度来说，动态规划往往可以用递归来实现，不过

确定了动态规划的这三要素，，表格需要（如最短路径、最长公共子序列、最大价值等），填表的过程就是根据递推关系，从1行1列开始，以行或者列优先的顺序，依次填写表格，最后根据整个表格的数据通过简单的取舍或者运算求得问题的最优解。

f(n,m)=max{f(n-1,m),f(n-1,w[n])+P(n,m)}

算法实现的步骤：

  1. 创建一个一维数组或者二位数组，保存每一个子问题的结果，具体根据题目而定。

    PS：需要创建二维数组的解法，都可以创建一个一维数组运用滚动数组的方式来解决，即一维数组中的值不停的变化

  2. 设置数组边界值，一维数组就是设置第一个数字，二维数组就是设置第一行和第一列的值，特变的滚动一维数组是要设置整个数组的值，然后根据后面不用的数据加进来变换成不同的值。

  3. 找出状态转换方程，也就是说找到每个状态跟他上一个状态的关系，根据状态转换方程写出代码。

  4. 返回需要的值，一般是一数组的最后一个或者是二维数组的最右下角。

例题分析

1、硬币问题

现有硬币若干，硬币的面值分别为1,5,11要凑出价值为w，至少需要多少枚硬币

输入格式：

​ 输入整数w

输出格式：

​ 输出最少需要多少枚硬币

输入输出样例：

输入：
	15
输出：
	3

解析：

​ 当要凑出价值价值w = 15，首先面临三种情况的选择

      1.    当选择了价值为1的硬币时，将面临凑出价格为14的情况
      2.    当选择了价值为5的硬币时，将面临凑出价格为10的情况
      3.    当选择了价值为11的硬币时，将面临凑出价格为4的情况

实现代码：

public static void main(String[] args) {
       int[] dp = new int[100];
       dp[0] = 0;
       for (int i = 1; i <= 15; i++) {
           int cost = Integer.MAX_VALUE;
           if(i-1>=0) cost = Math.min(cost,dp[i-1]+1);
           if(i-5>=0) cost = Math.min(cost,dp[i-5]+1);
           if(i-11>=0) cost = Math.min(cost,dp[i-11]+1);
           dp[i] = cost;
           System.out.printf("dp[%d]=%d\n",i,dp[i]);
       }
       System.out.println(dp[15]);
   }

原理：

​ f(n)只与f(n-1)，f(n-5)，f(n-11)有关

​ f(n) = min{1+f(n-1),1+f(n-5),1+f(n-11)}

总结：

​

​ 大问题的最优解可以由小问题的最优解推出

​

​ 一旦f(n)确定，那么之后直接调用它的值就可以，不需要再去关心f(n)的计算过程

2、01背包

现有四个物品，小偷背包总容量为8，怎么可以偷得价值最多的物品

如：

物品编号： 1 2 3 4

物品重量： 2 3 4 5

物品价值： 3 4 5 8

状态转移方程：

代码实现：

public static void main(String[] args) {
        int[][] f = new int[5][9];
        int[] w = {0,2,3,4,5};
        int[] v = {0,3,4,5,8};
        for (int i = 1; i < 5; i++) {
            for (int j = 1; j < 9; j++) {
                if(w[i]>j)//如果第i件装不下
                    f[i][j] = f[i-1][j];
                else //如果装得下，可以选择偷或不偷，取二者最大值
                    f[i][j] = Math.max(f[i-1][j],f[i-1][j-w[i]]+v[i]);
            }
        }
        System.out.println(f[4][8]);
    }

3、最长上升子序列(LIS):

给定长度为n的序列，从中选取一个子序列，这个子序列需要单调递增。

问最长上升子序列(LIS)的长度。

eg: 1,5,2,3,11,7,9

则LIS序列为：1,2,3,7,9，长度为5

输入格式：

​ 第一行输入n

​ 第二行输入n个数字

输出格式：

​ 序列的长度

代码实现：

static Scanner sc = new Scanner(System.in);
    public static void main(String[] args) {
        int n  = sc.nextInt();
        int a[] = new int[n+1];
        int dp[] = new int[100];
        for (int i = 1; i <= n; i++)
            a[i] = sc.nextInt();
        for (int i = 1; i <=n; i++) {
            dp[i] = 1;
            for (int j = 1; j < i; j++)
                if(a[j]<a[i])
                    dp[i] = Math.max(dp[i],dp[j]+1);
        }
        System.out.println(dp[n]);
    }

记f(x)为以a[x]结尾的LIS长度，那么LIS = max{f(x)}

考虑比x小的每一个p，如果a[x]>a[p]，那么f(x) = f(p)+1

f(x) = max{f(p)}+1

​ p<x,a[p]<a[x]

4、三角形最小路径和

给定一个三角形 triangle ，找出自顶向下的最小路径和。

每一步只能移动到下一行中相邻的结点上。相邻的结点 在这里指的是 下标 与 上一层结点下标 相同或者等于 上一层结点下标 + 1 的两个结点。也就是说，如果正位于当前行的下标 i ，那么下一步可以移动到下一行的下标 i 或 i + 1 。

示例 1：

输入：
5
7
3 5
8 1 0
2 7 8 4
4 5 2 6 9
输出：30

示例 2：

输入：
1 
-10
输出：-10

方法一：dfs

设计好dfs函数

void dfs(int x,int y,int cur)，其中，cur表示从起点(1,1)到达终点(x,y)的走过的路径和。

static void dfs(int x, int y , int cur){
    if(x==n){
        if(cur>ans)ans = cur;
        return;
    }
    //点(x,y)可以向左下和右下两个方向走
    dfs(x+1,y,cur+a[x+1][y]);
    dfs(x+1,y+1,cur+a[x+1][y+1]);
}

此种解法会超时

方法二：动态规划（自顶向下的思维方法：顺推法）

  1. 确定状态

    定义f[x][y]表示从(1,1)出发到(x,y)的路径最大权值和。
那么，最终答案ans=max{f[n][1],f[n][2],f[n][3],.....,f[n][n]}

  2. 状态转移方程

    f[x][y] = max{f[x-1][y-1],f[x-1][y]}+a[x][y];
边界条件：f[1][1] = a[1][1];

  3. 满足最优子结构

    显然，答案最终的最优解是由子问题的最优解推导而来，所以，复合最优子结构性质。

  4. 无后效性

    一旦某个状态f[x][y]确定，那么在之后的状态只需要调用它的值就可以了，而不用关心它是怎么来的，所以复合无后效性。

代码实现

static Scanner sc = new Scanner(System.in);
   public static void main(String[] args) {
       int n = sc.nextInt();
       int[][] a = new int[n+1][n+1];
       int[][] f = new int[n+1][n+1];
       for (int i = 1; i <= n; i++)
           for (int j = 1; j <= i ; j++)
               a[i][j] = sc.nextInt();
       f[1][1] = a[1][1];//边界
       //动态规划的顺推
       for (int i = 2; i <=n; i++)
           for (int j = 1; j <=n; j++)
               f[i][j] = Math.max(f[i-1][j],f[i-1][j-1])+a[i][j];
       int ans[] = f[n];
       Arrays.sort(ans);
       if(n==1)
           System.out.println(a[1][1]);
       else
           System.out.println(ans[n]);
   }

方法三：动态规划（自底向上的思维方法：逆推法）

  1. 确定状态

    定义f[x][y]表示从(x,y)出发到终点（第n行）的路径最大权值和。
那么，最终答案就是f[1][1]

  2. 状态转移方程

    f[x][y] = max{f[x+1][y],f[x+1][y+1]}+a[x][y];
边界条件：f[n][i] = a[n][i];(1<=i<=n)

  3. 满足最优子结构

    显然，答案最终的最优解是由子问题的最优解推导而来，所以，复合最优子结构性质。

  4. 无后效性

    一旦某个状态f[x][y]确定，那么在之后的状态只需要调用它的值就可以了，而不用关心它是怎么来的，所以复合无后效性。

代码实现

static Scanner sc = new Scanner(System.in);
    public static void main(String[] args) {
        int n = sc.nextInt();
        int[][] a = new int[n+1][n+1];
        int[][] f = new int[n+1][n+1];
        for (int i = 1; i <= n; i++)
            for (int j = 1; j <= i ; j++)
                a[i][j] = sc.nextInt();
        f[n] = a[n];
        //动态规划的逆推
        for (int i = n-1; i >=1 ; i--)
            for (int j = 1; j <=i ; j++)
                f[i][j] = Math.max(f[i+1][j],f[i+1][j+1])+a[i][j];
        System.out.println(f[1][1]);
    }
